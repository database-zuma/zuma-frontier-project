## What is Frontier?

**Frontier** is Zuma Indonesia's AI transformation initiative—a collection of intelligent systems designed to revolutionize retail operations from store floors to warehouse logistics.

We're not just building apps. We're building the **operational backbone of tomorrow**.

## Design Philosophy

> **Monochrome. Minimal. Meaningful.**

- **Color Palette:** Green-adjacent spectrum (`#00E273` → `#00D4AA` → `#00B899` → `#7AE060`)
- **Typography:** Plus Jakarta Sans (body) + Geist Mono (headings)
- **Aesthetic:** Japandi minimalism meets tech precision
- **Motion:** Subtle, purposeful Framer Motion animations

## Quick Start

```bash
# Clone the repo
git clone https://github.com/database-zuma/zuma-frontier-project.git
cd zuma-frontier-project/frontier-landing-page

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Deployment

Deployed on **Vercel** with automatic CI/CD from the `main` branch.

```bash
# Manual deploy
npx vercel --prod
```
