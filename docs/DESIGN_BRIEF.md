# Zuma Frontier Project - Landing Page Design Brief

## Project Overview

**Purpose:** Pitching visuals for Zuma Frontier Project (AI Adoption Initiative)
**Division:** Continuous Improvements Division, Operations Department  
**Target:** Leadership, stakeholders, department heads
**Goal:** Showcase innovation, drive AI adoption buy-in

---

## Design Direction

### Overall Aesthetic
- **Theme:** Modern tech-forward, minimal, elegant
- **Mood:** Innovation meets professionalism
- **Style:** Glassmorphism meets minimalism (clean but sophisticated)
- **Background:** Dark mode with subtle depth

### Key Visual Elements from References

#### 0. Primary Hero Treatment (CRITICAL)
**"FRONTIER" as the Star:**
- Word "FRONTIER" must dominate the hero section
- Contrast: "ZUMA" (light/thin) vs "FRONTIER" (bold/gradient)
- Visual hierarchy: FRONTIER > ZUMA > everything else
- Use gradient fill, glow effects, or 3D depth to make FRONTIER pop

#### 1. Font Inspiration (Kilo.ai)
- **Style:** Clean, modern sans-serif
- **Characteristics:** 
  - Excellent typographic hierarchy
  - Bold headlines with light body text
  - Clean geometric letterforms
- **Recommendation:** Inter, Geist, or Plus Jakarta Sans

#### 2. Minimalism Reference (Akor by Gavril)
- **Style:** Ultra-clean, generous whitespace
- **Key traits:**
  - Strong grid-based layout
  - Minimal decorative elements
  - Focus on content hierarchy
  - Subtle hover effects

#### 3. Big Header/Simplicity (Halo AI Studio)
- **Style:** Bold hero typography, centered layout
- **Key traits:**
  - Large, impactful headline
  - Subtle tagline/supporting text
  - Clean two-button CTA layout
  - Modern, AI-tech feel

#### 4. Glassmorphism Reference (Denis Concepts)
- **Style:** Subtle glass effects, depth layers
- **Key traits:**
  - Background blur effects
  - Semi-transparent overlays
  - Subtle gradients
  - Floating elements

#### 5. Navbar & Header (Visionforge Reference)
- **Layout:**
  - Logo top-left with beta/label badge
  - Navigation centered (Features, Resources, Examples, Pricing, About)
  - Auth buttons top-right (Sign in / Sign up with border)
  - Hero section below with massive typography
  - Floating UI elements (tooltips, badges)
  - "THE NEW ERA OF VISUALISATION" style headline treatment

## Typography Hierarchy

### Primary Font
- **Family:** Inter or Geist (modern geometric sans-serif)
- **Weights:** 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)

### Type Scale
```
H1 (Hero):     72px - 96px, Bold, Letter-spacing -2px, Line-height 1.0
H2 (Section):  48px - 56px, SemiBold, Letter-spacing -1px
H3 (Card):     24px - 32px, Medium
Body Large:    18px - 20px, Regular, Line-height 1.6
Body:          16px, Regular, Line-height 1.5
Caption:       14px, Regular
Small:         12px, Medium (labels, badges)
```

### Typography Treatment
- **Hero text:** Split styling - "ZUMA" in outline/thin + "FRONTIER" in solid bold
- **Gradient text:** `linear-gradient(135deg, #00E273 0%, #00FF88 100%)` for key words
- **Uppercase:** Section headers, badges, navigation

## Visual Effects & Animations

### 1. Glassmorphism Cards
```css
.glass-card {
  background: rgba(255, 255, 255, 0.03);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 16px;
}
```

### 2. Gradient Text
```css
.gradient-text {
  background: linear-gradient(135deg, #00E273 0%, #00FF88 100%);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

### 3. Floating Elements
- Subtle floating animation (up/down 10px)
- Duration: 3-4 seconds
- Easing: ease-in-out

### 4. Hover States
- Cards: Scale 1.02, border color change, shadow
- Buttons: Background fill, slight lift
- Links: Underline animation from left

### 5. Scroll Animations
- Fade up on scroll
- Staggered reveal for cards
- Smooth parallax on hero elements
